# Privacy Policy for TikTok API Uploader

## 1. Introduction
This Privacy Policy explains how TikTok API Uploader ("the App") collects, uses, and protects your information. The App is a local tool for uploading videos to TikTok via API.

## 2. Information We Collect
- **OAuth Tokens**: We collect access tokens from TikTok for API authentication. These are stored locally on your device.
- **Video Data**: The App processes video files you provide for upload, but does not store or transmit them beyond TikTok's API.
- **No Personal Data**: We do not collect personal information like names, emails, or browsing data.

## 3. How We Use Information
- Tokens are used solely for authenticating API requests to TikTok.
- Video uploads are handled directly via TikTok's servers.

## 4. Data Sharing
- Data is shared only with TikTok as required for API functionality.
- We do not sell or share data with third parties.

## 5. Data Security
- Tokens and files are handled securely in your local environment.
- We recommend using secure devices and not sharing tokens.

## 6. Your Rights
- You can revoke tokens via TikTok settings.
- Contact us for any privacy concerns.

## 7. Changes to Policy
We may update this Policy. Check the GitHub repository for updates.
